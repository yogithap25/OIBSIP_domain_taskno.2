Random Password Generator

Objective - To develop a graphical application that generates secure random passwords based on user-defined complexity requirements.

Steps Performed -
1. Designed a graphical user interface using Tkinter.
2. Added input controls to select password length.
3. Provided options to include uppercase letters, lowercase letters, digits, and symbols.
4. Implemented logic to generate random passwords following basic security rules.
5. Ensured at least one character from each selected category is included.
6. Integrated clipboard functionality to allow easy copying of generated passwords.

Tools Used - Python,Tkinter

Outcome -  A user-friendly GUI-based Random Password Generator that creates secure, customizable passwords and allows users to quickly copy them for immediate use.
